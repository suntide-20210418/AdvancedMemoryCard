# 高级内存卡

<ItemImage id="advanced_memory_card:advanced_memory_card" scale="4" />

高级内存卡是标准[内存卡](ae2:items-blocks-machines/memory_card.md)的增强版本，为Applied Energistics 2系统提供强大的自动化配置能力。它能够简化重复性的配置任务，并实现设备设置的高效批量部署。

与标准内存卡不同，高级内存卡具备多种操作模式和区域选择功能，使大规模AE2网络管理变得更加高效。

## 基本使用方法

高级内存卡的操作方式与普通内存卡类似：
- **Shift + 右键** 点击设备以复制其设置
- **右键** 点击设备以粘贴已存储的设置
- **Shift + 滚轮** 在不同模式间循环切换
- **对空右键** 打开配置GUI进行模式特定设置

# 操作模式

## 复制模式

复制模式是高级内存卡的主要功能。它允许您选择一个矩形区域，并将配置粘贴到该区域内所有兼容的设备上。

### 区域选择流程：
1. **第一点选择**：Shift + 右键点击所需区域的第一个角落
2. **第二点选择**：Shift + 右键点击对角线的相反角落
3. **批量粘贴**：在任意位置右键点击，将存储的配置粘贴到选定区域内的所有AE2设备

### 功能特点：
- 支持选择最大**2048个方块**体积的区域
- 适用于所有AE2方块实体（ME接口、存储总线、输出总线等）
- 兼容安装在线缆上的AE2部件（破坏平面、成型平面等）
- 可视化选择覆盖层用彩色边框显示选定区域
- 自动验证防止选择过大的区域

### 视觉指示器：
- **红色框**：已选择第一点，准备选择第二点
- **白色框**：已选择第二点，区域已定义
- **绿色框**：准备粘贴，区域已确认

## 配置模式

配置模式功能与标准内存卡完全相同，为单个设备提供基本的复制粘贴功能，不具备区域选择能力。

此模式适用于：
- 简单的一对一配置传输
- 需要传统内存卡行为时
- 微调单个设备设置时

# 高级功能

## 模式切换
按住卡片时使用**Shift + 鼠标滚轮**快速在模式间切换。当前模式将在聊天栏中显示。

## 配置界面
对空右键打开模式特定的配置界面，在此处您可以：
- 调整复制模式设置
- 配置区域选择参数
- 访问高级粘贴选项

## 兼容性
高级内存卡适用于所有支持内存卡操作的Applied Energistics 2设备和部件，包括但不限于：
- ME接口
- 存储总线
- 输入/输出总线
- 液位发射器
- P2P通道
- 破坏/成型平面
- 分子装配室

此工具显著减少了大规模AE2网络设置和维护所需的时间和精力。
