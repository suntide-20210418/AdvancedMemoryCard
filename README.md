# Advanced Memory Card 高级内存卡

![Minecraft Version](https://img.shields.io/badge/Minecraft-1.20.1-blue)
![Forge Version](https://img.shields.io/badge/Forge-47.4.10-orange)
![License](https://img.shields.io/badge/License-GPL--3.0-green)

一个Applied Energistics 2扩展模组，为AE2内存卡添加批量复制和高级配置功能。

## 🌟 功能特性

### 🔧 核心功能
- **批量复制模式** - 选择区域范围，一次性复制多个AE2设备的配置
- **配置模式** - 传统单设备配置功能
- **智能模式切换** - Shift+右键快速切换不同工作模式
- **可视化选择** - 直观的区域选择和状态提示

### 🎯 使用方法

#### 批量复制模式
1. 手持高级内存卡，Shift+右键复制单个AE2设备配置
2. 普通右键选择要复制的第一个角落位置
3. 再次右键选择对角位置，完成区域选择
4. 对着空气右键批量应用配置到区域内所有AE2设备

#### 模式切换
- **Shift+右键** - 切换模式

#### 配置模式
- 传统的AE2内存卡功能，用于单设备配置管理

## 📦 安装说明

### 前置要求
- Minecraft 1.20.1
- Forge 47.4.10 或更高版本
- Applied Energistics 2 (AE2) 模组

### 安装步骤
1. 下载最新版本的 [.jar](file://D:\mycode\ModDevelopment\forge-1.20.1-47.4.10-mdk\gradle\wrapper\gradle-wrapper.jar) 文件
2. 将文件放入 Minecraft 的 `mods` 文件夹
3. 启动游戏即可使用
